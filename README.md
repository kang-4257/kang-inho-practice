# team-02
Team 02 - MSP Architect Training 2026

Trivy와 Gemini AI를 활용한 지능형 보안 진단 및 자동 조치 인프라

Ubuntu/K3s 환경에서 보안 취약점을 감지만 하는 것이 아니라, Gemini AI를 통해 해결책을 스스로 제시하는 자가치유형 파이프라인을 구축

주요 기능 (Main Features)
- AI 자가치유: Trivy 스캔 결과를 AI가 분석하여 수정 코드 자동 생성.

- 논리적 망 분리: Web-DB 계층 간의 Zero-Trust 네트워크 정책 적용.

- 보안 자동화: 이미지 빌드 단계에서 고위험 취약점 선제적 차단.
